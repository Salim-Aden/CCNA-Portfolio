# 01 Network Fundamentals

## Overview

This section covers the foundational networking concepts required for CCNA certification. These labs establish core skills in subnetting, IP addressing, device initialization, and basic security configurations.
---

**Why These Matter:**

 Subnetting is the foundation of all IP networking
 Basic device configuration is the first step in any network deployment
 These skills are used daily by network engineers
---
## What's Inside

| Lab | Focus | Skills |
|-----|-------|--------|
| **01_Subnetting_Practice** | VLSM, IP addressing, router/switch/PC configuration | Subnet calculation, interface IP assignment, trunk/access ports, gateway configuration |
| **02_Basic_Device_Config** | Router initialization | Hostname, banner, console password, enable secret |
---

## Skills Demonstrated

| Category | Skills |
|----------|--------|
| **Subnetting** | VLSM, CIDR, subnet calculation |
| **Routing** | Router interface IP assignment |
| **Switching** | Trunk ports, access ports |
| **Security** | Console password, enable secret, MOTD banner |
| **Verification** | `show ip route`, `show interfaces trunk`, `ping` |
---

## Why These Labs

  Subnetting determines how IP addresses are allocated
  Device configuration is the first step in any network deployment
  Verification confirms everything works as designed
